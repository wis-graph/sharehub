## 빠른 테스트

### 1. GitHub Pages 접속
https://wis-graph.github.io/sharehub/ 접속

### 2. 기본 기능 테스트
- 사이드바가 보이는지 확인
- Home MOC가 로드되는지 확인
- 노트 링크가 작동하는지 확인
- 다크 모드 전환 테스트

### 3. 노트 렌더링 테스트
URL에 노트명 추가: `/note-name.md`
- 마크다운이 제대로 렌더링되는지 확인
- 코드 블록 하이라이팅 확인
- 내부 링크 동작 확인
